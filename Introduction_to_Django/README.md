# LibraryProject

LibraryProject is a Django-based application designed to manage a library's book collection, track lending records, and manage library users.

## Getting Started

### Prerequisites

- Python 3.x
- Django (you can install it with `pip install django`)
- MySQL (if using MySQL as your database)

### Setup

1. **Create a New Django Project**

   ```bash
   django-admin startproject LibraryProject
   ```
